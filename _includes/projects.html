    Launch Attack on DES Encryption
    • Information Security
    • January 2017 - Present
    In this project, aim is to understand the innards of Data Encryption Standard (DES). DES is the first symmetric key encryption standard adopted by NIST and US federal agencies including banking sector. Though DES is not the first or last block cipher it definitely is computer science’s first step towards increasing security by using cryptography. The first block cipher happens to be Lucifer, introduced by IBM and is a direct precursor to DES. Thus, DES is kind of influenced by the design of Lucifer. Built the original code for DES Encryption. After this design of launching an attack on DES code breaking using known plain text brute force attack is required. For this phase “plain text” and "cipher text” is needed. The attack must find the “key” used to perform this encryption.
